* **Матрица клиентских ошибок (Response Errors 4xx):**

  * **Ошибка 400 Bad Request (Неверный формат параметра)**
    * *Описание*: Параметр `order_id` передан в нетекстовом или невалидном формате (ожидалось целое число `INT`).
    * *BODY:*
      ```json
      {
        "code": 1001,
        "message": "order_id: ожидалось целое число (INT)"
      }
      ```

  * **Ошибка 401 Unauthorized (Отсутствие авторизации)**
    * *Описание*: В заголовках запроса (Headers) не передан токен авторизации или он протух.
    * *BODY:*
      ```json
      {
        "code": 2001,
        "message": "Пользователь не авторизован в системе Континент"
      }
      ```

  * **Ошибка 404 Not Found (Заказ не существует)**
    * *Описание*: Запрос прошел авторизацию, но указанного `order_id` нет в базе данных аптеки.
    * *BODY:*
      ```json
      {
        "code": 4004,
        "message": "Заказ с указанным идентификатором не найден"
      }
      ```
