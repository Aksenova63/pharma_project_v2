# ТЕХНИЧЕСКОЕ ЗАДАНИЕ: ПЛАТФОРМА ЛОЯЛЬНОСТИ И ОНЛАЙН-ЗАКАЗОВ «КОНТИНЕНТ»

## 1. БИЗНЕС-ТРЕБОВАНИЯ (BUSINESS REQUIREMENTS DOCUMENT - BRD)

### 1.1. Бизнес-цели проекта:
* **Удержание клиентской базы**: Снижение оттока клиентов к конкурентам за счет предоставления сервиса гарантированного наличия лекарственных средств.
* **Перевод продаж в онлайн-канал**: Запуск мобильного приложения для обеспечения возможности предварительного заказа.
* **Оптимизация складских остатков**: Получение данных о спросе до физического визита клиента в аптеку.

### 1.2. Ключевые функции системы (Scope):
1. **Модуль онлайн-резервирования**: Возможность оформления заказа через мобильное приложение с холдированием товара на складе конкретной аптеки.
2. **Модуль программы лояльности**: Начисление и списание бонусных баллов за выкупленные заказы.
